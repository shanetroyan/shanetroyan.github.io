Work Order Application

Work Order Application Compiled in VB.NET and ASP using WinForms platform. This application allowes the user to create, remove, update and delete Employees, Customers, Dealers, and Vendors from the stored database (PostgreSQL DBMS). Additionally the user can backup the database to a local source (such as the computer's hard drive). The login screen requires an employee initial populated from the drop-down list and a valid password. If the user is an administrator they have the ability to remove records such as the customers, employees, vendors and shippers. The user also has the ability to print, and export records to a csv file if they choose. Bi-Weekly Backups were being worked on but never fully implemented to ping a remote source. (DBMS). This application was modeled on the Microsoft Access program that my uncle Carl Huben had written in Microsoft Access 1997.
